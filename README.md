# SerialOut
Beat saber mod that sends light events to serialport. Events can be used by a controller (ESP32, Arduino). Chroma is also supported.

## Requirements:
	BSIPA
	CustomJSONData
	BeatSaberMarkupLanguage
	BS Utils

[Tutorial](https://github.com/Felix212/SerialOut/wiki)
